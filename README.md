# 📊 Credit Risk Prediction using Machine Learning

## 🎯 Problem Statement

Financial institutions need to assess whether a customer is likely to default on a loan.
This project builds a machine learning model to classify customers as **risky or safe**.

---

## 🧠 Approach

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Train-Test Split
* Random Forest Classifier
* Handling Imbalanced Data using `class_weight`
* Model Evaluation (Precision, Recall, F1-score)

---

## 📈 Results

* Accuracy: ~76%
* Improved detection of risky customers using balanced model

---

## 🔍 Explainable AI (SHAP)

Used SHAP to:

* Identify most important features
* Explain model predictions
* Understand how each feature impacts risk

---

## 💡 Key Insights

* Loan duration significantly affects credit risk
* Account status plays a major role in prediction
* Customers with lower values in key features are more likely to default

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib
* SHAP

---

## 📌 Conclusion

Built an end-to-end machine learning pipeline and applied Explainable AI to make model predictions transparent and interpretable.
