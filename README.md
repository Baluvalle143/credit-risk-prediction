# 📊 Credit Risk Prediction using Machine Learning

## 🎯 Problem
Banks need to decide whether a customer is risky or not before giving a loan.  
This project predicts credit risk using machine learning.

---

## 🧠 Approach
- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Train-Test Split  
- Random Forest Model  
- Handling Imbalanced Data using class_weight  
- Model Evaluation (Precision, Recall, F1-score)  

---

## 📈 Results
- Accuracy: ~76%  
- Improved detection of risky customers  

---

## 🔍 Explainable AI (SHAP)
Used SHAP to:
- Identify important features  
- Explain model predictions  
- Understand feature impact on risk  

---

## 💡 Key Insights
- Loan duration affects credit risk  
- Account status plays major role  
- Low-value features increase default probability  

---

## 🛠️ Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib  
- SHAP  

---

## 📌 Conclusion
Built an end-to-end machine learning pipeline and interpreted results using explainable AI.
